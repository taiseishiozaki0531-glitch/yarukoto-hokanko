# Product Overview

最終更新: 2026-05-13  
更新理由: Bootstrap 時の汎用説明を、ユーザー提供の「やること保管庫」プロダクト方針に同期。

「やること保管庫」は、読書・動画・勉強・人間関係・買い物など、日常生活で「あとでやろう」と思ったまま放置されがちなものを一箇所に保存し、「次にやること」まで整理する個人向け Web アプリです。

単なる ToDo リストではなく、明確な締切がないために放置されやすい「後回し案件」を管理することを目的とします。タスク完了だけでなく、未着手のまま頭の片隅に残るものを回収し、次の小さな行動へつなげることを重視します。

## Target Users

主な対象ユーザーは、大学生・高校生・若手社会人など、学習・趣味・人間関係・買い物などの小さな後回し案件を多く抱えている個人ユーザーです。

複数人での業務管理よりも、個人が自分の関心・予定・保留事項を整理し、日々見返して進められる体験を優先します。

## Core Capabilities

初期リリースでは、個人利用向けのレスポンシブ Web アプリとして以下の体験を提供します。

- ユーザー認証により、自分のアイテムだけを扱える
- アイテムを登録し、一覧・詳細・編集・削除できる
- カテゴリ・ステータス・優先度でアイテムを絞り込める
- ダッシュボードで全体状況、進捗率、期限が近いものを把握できる
- 「今日やるリスト」で次に取り組むものを確認できる
- URL を持つアイテムは、すぐに元の読書・動画・勉強・商品ページなどを開ける
- 期限切れ、今週中に期限が来るアイテム、空状態をわかりやすく表示する

## Initial Release Boundaries

初期リリースでは、後回し案件の保存・整理・見返し・次アクション化に集中します。

実装しないもの:

- AI 分類
- 通知
- 外部 API 連携
- 共有機能
- 画像アップロード

これらは将来の拡張候補ですが、初期リリースの仕様やタスクに混ぜません。必要になった場合は別 spec として境界を切ります。

## Value Proposition

「やること保管庫」の価値は、締切のある ToDo では拾いにくい「いつかやる」を失くさず、次に何をすればよいかまで軽く整理できることです。

読書・動画・勉強・人間関係・買い物を別々の仕組みに分けず、共通のアイテムとして扱うことで、ユーザーは生活の中に散らばった後回し案件を一箇所で見渡せます。
